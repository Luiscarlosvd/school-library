# 📖 Create Enumerable <a name="about-project"></a>

School Library app that allows user to add new students or teachers, add new books and save records of who borrowed a given book and when. Built with Ruby.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

  <ul>
    <li><a href="https://www.ruby-lang.org/en/">Ruby</a></li>
  </ul>

<!-- Features -->

### Key Features <a name="key-features"></a>

- Create person class.
- Create student and classroom classes.
- Create teacher class.
- Create book and rental classes.
- Create basic UI.

<!-- GETTING STARTED -->

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

1. Code editor.
2. Git.
3. [Ruby](https://www.ruby-lang.org/en/)

### Setup

In order to clone this repository to your desired folder, run the commands below on your console:

```sh
    cd my_folder
    git clone https://github.com/Luiscarlosvd/school-library.git
```

### Install

Install this project with:

```sh
  bundle install
```

### Usage

To run the project, execute the following command:

```sh
  ruby person.rb
```
